# **End-to-End-Sentiment-Analysis**

## How to Run ?
### Clone Project
```bash
git clone https://github.com/your-repo/Network-Security-Project.git
cd Network-Security-Project
```

### Create Env
```bash
conda create -p env python==3.11 -y

conda activate ./env
```
### Install Dependencis
```bash
pip install -r requirements.txt
```
### Run app.py
```bash
python app.py
```
